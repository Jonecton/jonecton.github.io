Projects

SecureLens - Elementary School Security

SecureLens is an innovative security solution designed for elementary schools. It features automatic license plate recognition (ALPR) software that monitors traffic and reads license plates using OpenCV and EasyOCR. Developed in Python with the Flask framework, SecureLens utilizes SQLite3 to manage the database.

The primary function of SecureLens is to enhance the security and efficiency of student pick-up procedures. It accomplishes this by comparing license plates with a database of registered students. When a recognized license plate is detected, it automates the release of students from their classrooms by displaying their names on a classroom screen.

SecureLens also provides students with unique ID cards that can be scanned by teachers at the pickup area. After scanning, essential information, including the student's name, parent names, the color, make, and model of the recognized vehicle, and the plate number, is displayed on an Android device, allowing teachers to verify that the individual picking up the child is authorized to do so.

Additionally, SecureLens offers a blacklist feature, allowing schools to identify and flag restricted license plates. When a restricted plate is recognized, campus authorities are alerted, enabling them to implement any necessary security measures.
